# 📊 Vrinda Store Annual Sales Dashboard (Excel)

## 📌 Project Overview

The **Vrinda Store Annual Sales Dashboard** is an interactive Excel dashboard created to analyze the store's sales performance for the year 2022. The dashboard provides insights into sales trends, order status, customer demographics, regional performance, and sales channels through dynamic visualizations and slicers.

This project demonstrates data analysis and dashboard-building skills using Microsoft Excel.

---

## 🎯 Objectives

- Analyze annual sales performance.
- Track monthly sales and order trends.
- Understand customer demographics.
- Identify top-performing states.
- Evaluate sales channel performance.
- Support business decision-making with data-driven insights.

---

## 📈 Dashboard Features

### 🔹 Order Status Analysis
- Delivered Orders: 92%
- Cancelled Orders: 3%
- Returned Orders: 3%
- Refunded Orders: 2%

### 🔹 Orders vs Sales Trend
- Monthly sales amount analysis.
- Monthly order count comparison.
- Identification of peak sales periods.

### 🔹 Customer Demographics
- Gender-wise analysis (Men vs Women)
- Age Group Analysis:
  - Adult
  - Teenager
  - Senior

### 🔹 Top Performing States
- Maharashtra
- Karnataka
- Uttar Pradesh
- Telangana
- Tamil Nadu

### 🔹 Sales Channel Analysis
- Amazon
- Flipkart
- Myntra
- Ajio
- Meesho
- Nalli
- Others

### 🔹 Interactive Filters
- Month-wise filtering
- Category-wise filtering
- Channel-wise filtering

## 📊 Key Insights

- Women customers contributed a larger share of orders than men.
- Adults were the most active customer segment.
- Amazon and Flipkart generated the highest order volumes.
- Maharashtra recorded the highest sales revenue.
- Most orders were successfully delivered, indicating strong operational performance.

---

## 🚀 How to Use

1. Download the Excel workbook.
2. Open it in Microsoft Excel (2016 or later).
3. Use the slicers to filter data by Month, Category, and Channel.
4. Explore the dashboard and insights.
